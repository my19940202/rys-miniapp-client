# 教育行业的小程序

## C端：小程序
### 地图展示
  - 支持景点标注（如山体、古建筑、厕所、服务点等）
  - 地图支持手势缩放和拖动操作
  - 支持GPS定位当前位置
### 路线规划
  - 可设定推荐路线（例如从北门进的路线和从南门进的游览路线）
  - 支持"从当前位置导航到某一景点"的简单路径规划（步行路线，基于地图图层）
### GPS定位功能
  - 支持 GPS 定位
  - 在地图中显示当前位置（如当前在游客中心）
###  导览介绍模块
- 景点信息展示
  - 地图上点击，景点图标点击后弹出信息框
  - 信息框包含：景点名称、图片、简介、语音导览按钮
- 语音介绍播放
  - 每个地标点配套一段语音讲解（普通话）
  - 支持播放、暂停、快进、重听功能
